# Keystones CAD

Luxury one-page marketing website for Keystones CAD, built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and GSAP.

## Positioning

Keystones CAD provides professional jewelry CAD design services for jewelers, jewelry manufacturers, jewelry brands, and wholesalers worldwide.

- We do not sell jewelry.
- We do not manufacture jewelry.
- We provide CAD design services and production-ready digital deliverables only.

## Included Deliverables Messaging

- STL Files
- 3DM Files
- 360° Product Videos
- High-End Renders
- Wax Models
- Production-Ready CAD Files
- Additional manufacturing files as required

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
